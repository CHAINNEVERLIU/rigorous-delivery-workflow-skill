# Skill Selection

This file is kept for backward compatibility. Use `skill-routing-matrix.md` as the authoritative routing guide.

Minimum rule:
- load the smallest useful set of skills/tools,
- do not invent unavailable tools,
- record manual equivalents when a matching skill is unavailable,
- use subagents only through `subagent-protocol.md`,
- use external AI only through `external-ai-handoff.md`.

See `skill-routing-matrix.md`.
